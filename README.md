# :fire::fire: Dracarys :fire::fire:

A simple React Native CLI to generate

* Function Components
* Class Components
* Styles
* Redux Sagas
* Redux module

## Installation

Install it globally
```bash
yarn global add dracarys-cli
```

## Usage

Just dracarys :fire: and follow the steps

```bash
dracarys
```
